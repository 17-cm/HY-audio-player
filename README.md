# 🎵 HY 音乐播放器

一个为 SillyTavern 设计的功能强大的音乐播放器扩展，支持网易云音乐直链导入、歌单批量导入、多种播放模式和视觉效果。

> **重要声明**：本扩展完全免费，禁止任何形式的收费分发。

---

## ✨ 主要特性

- 🎶 多种播放模式（列表顺序、单曲循环、列表随机）
- ☁️ 网易云音乐支持（单曲和歌单批量导入）
- 🎨 自定义外观（RGB灯效、磨砂玻璃、自定义颜色）
- 📱 完美支持移动端触摸操作
- 🎭 四种显示模式（窄屏、全屏、律动、纯享）
- 🎤 LRC格式歌词显示，支持渐变色
- 💾 自动保存播放列表和设置
- 🔄 播放器和律动图标可自由拖动

---

## 📦 安装方法

### 通过 Git 地址安装

1. 打开 SillyTavern
2. 进入 **扩展（Extensions）** 页面
3. 输入：`https://github.com/17-cm/HY-audio-player.git`
4. 点击安装

---

## 🎮 快速开始

### 基本操作

- **显示播放器**：点击扩展设置中的悬浮图标，或在扩展设置中开启
- **添加歌曲**：点击"+"按钮，支持本地文件、直链、网易云链接
- **拖动定位**：点击顶部灵动岛拖动播放器位置
- **切换模式**：点击右侧按钮切换不同显示模式

### 快捷键（电脑端）

- `Alt + P`：播放/暂停
- `Alt + N`：下一首
- `Alt + B`：上一首
- `空格`：播放/暂停

### 播放器模式

1. **窄屏模式**：默认紧凑显示
2. **全屏模式**：展开设置和播放列表
3. **律动模式**：最小化为律动波形图标
4. **纯享模式**：全屏歌词显示

---

## ☁️ 网易云音乐支持

支持以下格式的链接：
- `music.163.com/song?id=xxx`（单曲）
- `music.163.com/playlist?id=xxx`（歌单）
- `y.music.163.com/m/song/xxx`（移动端链接）
- `163cn.tv/xxx`（短链接）

功能：
- 自动获取歌曲信息、封面、歌词
- 支持歌单批量导入
- 导入历史记录

---

## ⚙️ 自定义设置

### 外观设置
- 播放器尺寸、边框宽度和颜色
- 封面尺寸调整
- RGB灯效（关闭/单色/幻彩）
- 字体颜色自定义

### 背景设置
- 磨砂玻璃效果和透明度
- 全屏/窄屏背景颜色
- 支持上传背景图片

### 歌词设置
- 渐变色起始/结束色
- 纯享模式歌词显示效果

### 扩展设置
- 悬浮图标开关
- 图标颜色自定义
- 自动适配主题
